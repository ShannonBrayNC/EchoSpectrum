# Observatory UI

The EchoSpectrum Observatory UI provides a live receive-only RF visualization experience.

## Initial Features

- synthetic FFT visualization
- live spectrum scope
- compliance status panel
- session metadata display
- observatory aesthetic
- mobile-friendly layout foundation

## Planned Features

- waterfall rendering
- WebSocket streaming
- GPS overlays
- directional antenna visualization
- AI signal tagging
- replay timeline
- distributed sensor node dashboards

## Technology Stack

- React
- TypeScript
- Vite
- Recharts
- Tailwind CSS (future enhancement)

## Governance Requirements

The UI must:
- display RX-only compliance state
- avoid offensive RF functionality
- expose governance metadata
- surface ETS policy warnings
